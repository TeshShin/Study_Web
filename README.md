## CSS의 Selectors
1. class는 .선택자이름{} 으로 만들고 그가 가진 선언에 따라 class="선택자이름" 을 가진 태그들이 영향을 받는다.
2. ID는 #선택자이름{}, ID="선택자이름"
3. 우선순위 ID > class > 일반태그 이다.
